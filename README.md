# Blackjack-GAME - Enhanced

A browser-based Blackjack game built with HTML, CSS, and JavaScript.

## Live Demo

https://harshbakraniya.github.io/Blackjack-GAME-Enhanced/

## Project Background

This project began as a basic Blackjack tutorial project. After completing the tutorial version, I continued developing it independently and added several new gameplay and user interface features.

The goal of this version was to practice working with game state, user interactions, DOM manipulation, and feature development beyond the original tutorial.

### Improvements Added

* Betting system ($10, $20, $30, and $50 bets)
* Chip balance management
* Rules popup for first-time players
* Dynamic betting interface
* Win/Loss chip calculations
* Restart game flow improvements
* User interface enhancements

## How to Play

1. Read the game rules.
2. Select a bet amount.
3. Click START GAME.
4. Draw cards using NEW CARD.
5. Reach 21 without going over.
6. Win chips by getting Blackjack.
